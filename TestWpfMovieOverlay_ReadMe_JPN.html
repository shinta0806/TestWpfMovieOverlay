概要

.NET Framework C# WPF (Windows Presentation Foundation) による、動画再生時に動画にテキストや図を上書きするサンプルコードです。

動作環境

以下の環境で動作することを確認しています。再生できる動画の形式は環境やインストールされているコーデックにより異なるかもしれませんので、標準の Windows Media Player で再生できる動画でお試し下さい。

  • Windows 10 Pro（1903）
  • .NET Framework 4.5
  • Visual Studio 2019 Community エディッション
  • 映像 H.264 (AVC)、音声 AAC の MP4 動画

詳細

詳細については、ブログの記事

をご覧ください。

ビルド方法

Visual Studio 2019 でソリューションファイル（TestWpfMovieOverlay.sln）を開き、ビルドしてください。
